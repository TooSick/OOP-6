Для рисования фигур необходимо зажать левую кнопку мыши(пока она зажата доступен предпросмотр фигур).
Чтобы закончить рисование необходимо отпустить левую кнопку мыши(для линий, прямоугольников и элипсов).
Для рисования ломанной и многоугольника необходимо выполнить теже этапы, но для окончания рисования
необходимо нажать правую кнопку мыши.
